yy-fps

yy-fps

Yet another vanilla FPS indicator with graph

features
  • simple one call per loop
  • fully customizeable and styleable

installation

 
npm i yy-fps

simple example

 
const FPS = require('yy-fps')
const Loop = require('yy-loop')

const fps = new FPS()

// start a loop
const loop = new Loop()
loop.add(update)
loop.start()

// update function
function update()
{
    fps.frame()

    // do stuff like rendering and dancing
}
